## 二、奥特曼等级系统

### 2.1 等级划分

升级需要**识字量和古诗词背诵量同时满足**：

| 等级 | 奥特曼 | 识字量要求 | 古诗词要求 | 等级称号 |
|------|--------|-----------|-----------|----------|
| 起始 | 捷德 | — | — | 新生代英雄 |
| Lv.2 | 银河 | 200 字 | 5 首 | 未来之光 |
| Lv.3 | 赛罗 | 300 字 | 10 首 | 最强战士 |
| Lv.4 | 艾斯 | 400 字 | 15 首 | 光线王 |
| Lv.5 | 奥特之父 | 600 字 | 20 首 | 光之国最高领袖 |

**说明**：
- 捷德是起始等级，不需要任何条件
- 识字量和古诗词背诵量**都满足**才能升级
- 等级提升时触发庆祝动画（光芒+音效）

### 2.2 等级展示

- **训练基地页（主页）**：顶部统计栏显示豆包头像缩略图（doubao.png）+ 当前等级角标，不占主要空间
- **荣耀殿**：完整展示豆包头像（大图）+ 等级角标 + 等级名称 + 进度条，进度分两行显示：
  > 已识 **128** 字 · 已背 **6** 首古诗
  > 距升级还需：再识 72 字、再背 4 首诗
- 满级（奥特之父）进度条显示「已达最高等级！」
- 头像边框颜色随等级变化（跟随奥特曼主题色）

## 三、训练基地页面

### 3.1 Tab 切换

页面顶部有两个 Tab：**识字** | **古诗词**，点击切换内容区域，底部导航不动。

### 3.2 识字 Tab

- 与原「识字记录」页功能完全一致
- 输入今天学的字，查看最近学习记录

### 3.3 古诗词 Tab

**添加诗词**：
- 输入字段：诗名、作者、朝代、诗句（逐行输入或整段粘贴后自动按行分割）
- 保存后出现在「最近添加」列表

**最近添加列表**：
- 显示最近几首诗，格式：📜 静夜思 · 李白
- 点击可查看全文

## 四、百宝箱页面

### 4.1 Tab 切换

页面顶部有两个 Tab：**汉字** | **诗词**

### 4.2 汉字 Tab

与原「字库」功能完全一致（搜索、导出、导入、排序）

### 4.3 诗词 Tab

- 所有诗词列表，可搜索
- 每首诗显示：诗名、作者、状态（学习中 / 已背诵）
- 点击查看全文
- 可删除诗词

---

## 五、闯关页面

### 5.1 测试类型选择

进入闯关后先选择：**识字** | **古诗词**

### 5.2 识字闯关

与原测试功能一致，可选 5 / 10 / 20 题

### 5.3 古诗词闯关

- 可选题数：**5 题** 或 **10 题**
- 题型：**随机**选择「接下一句」或「补充上一句」
  - **接下一句**：显示某句，判断/填写下一句
  - **补充上一句**：显示某句，判断/填写上一句
- 测试方式：展示一句诗，豆包判断自己会不会（✅ 会 / ❌ 不会）
- 满分（全部会）触发卡片奖励（**独立**每日1张限制，与识字闯关分开计算）

### 8.2 等级计算逻辑

```javascript
// 同时满足识字量和古诗词数量才能升级
function getUltramanLevel(wordCount, memorizedPoemCount) {
  if (wordCount >= 600 && memorizedPoemCount >= 20) return 'father';
  if (wordCount >= 400 && memorizedPoemCount >= 15) return 'ace';
  if (wordCount >= 300 && memorizedPoemCount >= 10) return 'zero';
  if (wordCount >= 200 && memorizedPoemCount >= 5)  return 'ginga';
  return 'geed';
}
```

### 8.3 旧数据迁移策略

- 从云端加载旧 `profile` 后，直接用新字段默认值覆盖缺失字段
- `poems` 字段不存在时初始化为空数组
- `currentLevel` 始终根据当前识字量 + 已背诵诗词数重新计算

## 十、业务逻辑

### 10.1 连续学习定义

- **学习行为**：添加字 OR 添加古诗 OR 完成任意测试（任一即算）
- **计算方式**：按自然日计算，每日至少完成一次学习行为
- **断签处理**：中断后重置为 0

### 10.2 卡片获取边界情况

- 当天已通过识字闯关获得卡片，再通过古诗词闯关满分 → 提示「今日卡片已获取，明天再来！」
- 识字量达到里程碑（如 300 字）→ 立即弹出卡片，不受每日限制
- 识字量里程碑与升级同时触发时 → 先播升级动画并跳转荣耀殿，跳转完成后再弹发卡弹窗

### 10.3 等级更新逻辑

- 每次添加字、添加诗词、完成古诗词测试后检查是否达到下一等级阈值
- 升级时触发庆祝动画（全屏12条光芒射线 + 豆包头像浮动放大 + 等级角标）
- 动画结束后自动跳转到荣耀殿

### 10.4 古诗词背诵状态

- 初始状态为「学习中」
- 古诗词闯关满分后，该诗状态变为「已背诵」
- 已背诵的诗词数量用于计算奥特曼等级
